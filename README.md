# Movie Planner Bot — Официальный сайт

Официальный сайт сервиса Movie Planner Bot — планировщика кино для Telegram.

## Путь к сайту на компьютере

```
/Users/nikitazaporohzets/Desktop/scripts/movie_planner_site/
```

## Описание

Сайт с лендингом, личным кабинетом (read-only для пользователей с базой и онбордингом для пустой базы), входом по коду из бота и разделами: Планы, Непросмотренные, Сериалы, Оценки, О проекте, FAQ.

## Структура

- `index.html` — основная страница, шапка, модалка входа, экраны кабинета
- `style.css` — стили в стилистике браузерного расширения
- `app.js` — логика входа, API, отображение кабинета
- `images/` — скриншоты бота

## Фото в разделе «О проекте»

Чтобы отобразить своё фото в блоке «О проекте»:

1. Сохраните изображение как **`images/author.png`** (или `images/author.jpg`) в папке `images/` рядом со скриншотами.
2. Рекомендуемый размер: **квадрат 300×300 или 400×400 px** (на сайте оно обрежется в круг).
3. Формат: JPG или PNG.

Если файла нет, блок с фото просто не показывается.

## Деплой

Сайт готов к деплою на GitHub Pages, Netlify, Vercel или любой другой хостинг статических сайтов. Для работы личного кабинета нужен бэкенд Movie Planner Bot с API `/api/site/*`.
