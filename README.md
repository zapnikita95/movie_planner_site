# Movie Planner Bot — Официальный сайт

Официальный сайт сервиса Movie Planner Bot — планировщика кино для Telegram.

## Описание

Информационный one-page сайт для Movie Planner Bot с описанием функционала, скриншотами и ссылками на бота и браузерное расширение.

## Структура

- `index.html` — основная страница
- `style.css` — стили в стилистике браузерного расширения
- `images/` — скриншоты бота

## Деплой

Сайт готов к деплою на GitHub Pages, Netlify, Vercel или любой другой хостинг статических сайтов.
