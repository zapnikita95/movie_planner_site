MCP и AI-агенты: подключите нейросеть к Movie Planner

29 мая 2026

Обычный чат с ИИ умеет советовать фильмы — и забывает через пять минут. Movie Planner хранит вашу базу, планы, билеты и напоминания. А через MCP (Model Context Protocol) вы подключаете агента на своём компьютере — Claude Desktop, Cursor, другие IDE — и просто говорите: «добавь «Дюну» в календарь на субботу» или «прикрепи билет к плану». Агент вызывает инструменты Movie Planner, и действие реально выполняется.

Что такое MCP и зачем он пользователям Movie Planner

MCP — открытый протокол, через который нейросеть получает доступ к внешним сервисам как к «инструментам». Вместо копирования ссылок и ручного планирования вы формулируете задачу на человеческом языке, а агент дергает API Movie Planner.

Это не эксперимент «на будущее»: MCP-сервер Movie Planner уже работает — локально (stdio) и по HTTP на бэкенде. Через полгода–год такой сценарий станет стандартом для любого персонального ассистента. Сейчас — редкое преимущество для тех, кто настроит интеграцию первым.

Что можно поручить агенту

Примеры фраз, которые превращаются в реальные действия:

  • «Добавь фильм Матрица в мою базу» — фильм появится в watchlist;
  • «Запланируй просмотр «Опенгеймер» в кино в субботу в 19:00, кинотеатр Октябрь» — создаётся план с напоминанием;
  • «Добавь это событие в Google Календарь» — пуш в календарь или ссылка на .ics;
  • «Вот скрин билета — прикрепи к плану на воскресенье» — билет сохраняется и придёт в уведомлении за 10 минут до сеанса;
  • «Что у меня в непросмотренном?» — агент читает вашу библиотеку и отвечает по фактам;
  • «Создай группу для друзей и дай invite-ссылку» — общий watchlist для компании;
  • «Запусти Interstellar на телевизоре» — команда на Smart TV через TV-агент;
  • «Поставь оценку 9 фильму, который смотрели вчера» — через API кабинета (расширяется в MCP).

ИИ перестаёт быть собеседником и становится исполнителем: план попадает в базу, напоминание придёт в Telegram или приложение, билет не потеряется в галерее.

Какие инструменты доступны агенту

Официальный MCP-сервер Movie Planner оборачивает версионированный HTTP API /v1. Основные инструменты:

  • Фильмы — добавление по ссылке Кинопоиска или kp_id;
  • Планы — домашний просмотр или поход в кино с датой, временем и кинотеатром;
  • Календарь — Google Calendar, подписанные .ics-ссылки;
  • Билеты — загрузка изображения к кино-плану, чтение списка;
  • Библиотека — непросмотренное, профиль пользователя;
  • Группы — создание и invite-ссылки для друзей;
  • Smart TV — запуск фильма на телевизоре, поиск на Rutube/VK.

Полный контракт: OpenAPI, обзор для разработчиков — Нейросети и API.

Как подключить (Cursor, Claude Desktop)

  1. Войдите в Movie Planner — веб-кабинет или приложение для iPhone.
  2. Получите Bearer-токен: войдите в кабинет → раздел «Нейросети и API», или пройдите OAuth2 PKCE для внешнего приложения (/developer).
  3. Добавьте MCP-сервер в конфиг IDE. Пример для Cursor:
{
  "mcpServers": {
    "movie-planner": {
      "command": "python3",
      "args": ["-m", "moviebot.mcp_server.stdio_server"],
      "cwd": "/path/to/movie_planner_bot",
      "env": {
        "MP_BASE_URL": "https://api.movie-planner.ru",
        "MP_BEARER_TOKEN": "ваш_токен"
      }
    }
  }
}

Токен храните в секретах IDE, не коммитьте в git. Альтернатива — HTTP MCP на api.movie-planner.ru/mcp с тем же Bearer в заголовке Authorization.

Почему это сильнее обычного GPT

ChatGPT без интеграции не помнит ваш watchlist через месяц, не пришлёт билет в нужный момент и не синхронизирует план с друзьями. Movie Planner + MCP даёт:

  • персистентную память — база фильмов и сериалов живёт годами;
  • напоминания — push в приложение и на сайт, не «напомни мне в чате»;
  • группы — один план видят все участники;
  • билеты и календарь — исполнение, а не текст «я бы добавил в календарь»;
  • Smart TV — запуск на телевизоре одной командой агенту.

Безопасность

Агент действует от вашего имени только с вашим токеном. OAuth2 поддерживает scopes: можно выдать доступ только на чтение планов или только на добавление фильмов. Токен можно отозвать в любой момент в кабинете.
